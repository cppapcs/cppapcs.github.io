Directions for 2017 Prep Course

Implement interesting projects while focusing on CS AP exam material

Industry Seasoned Professors

Learn from faculty of the Cal Poly Pomona Computer Science Department. Instructors have stories and lessons from industry to share!

Computing with Fun

Real World Projects

Learn Programming in a group setting, hands-on lab environment while preparing for the AP CS Exam. Many books tell you what to learn to pass the exam, but not WHY learn those topics. We provide you opportunity to work with real world projects which will not only motivate you to learn better but also apply it in future. Some of the projects include:

  • Artificial Intelligence and Robotics
  • Network Programming and Online Chat
  • Cyber Security and Game Development

Learn Java Programming Language

Programming language is the communication medium that helps human being to tell the computer what to do.
